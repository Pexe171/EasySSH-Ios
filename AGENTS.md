# AGENTS.md

## Regras locais do projeto

- Sempre versionar mudancas relevantes com commit antes de encerrar uma entrega.
- Antes de editar, verificar `git status` para nao sobrescrever trabalho em andamento.
- Nunca reverter alteracoes existentes sem pedido explicito.
- Depois de alterar codigo Android, validar com `testDebugUnitTest` e `assembleDebug` sempre que viavel.
- Se uma mudanca quebrar o app, voltar para o commit anterior e refazer com escopo menor.
